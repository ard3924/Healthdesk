# Health Desk Clinic

A modern, responsive e-commerce frontend for Health Desk Clinic, specializing in all-natural health supplements. Built with React, Vite, and Tailwind CSS to provide a seamless shopping experience for wellness products.

## Overview

Health Desk Clinic offers a comprehensive range of science-backed, all-natural supplements designed to support holistic health and wellness. Our platform features carefully curated products across multiple health categories, backed by research and manufactured with eco-conscious practices.

## Installation

1. **Clone the repository**:

   ```bash
   git clone <repository-url>
   cd health-desk-clinic
   ```

2. **Navigate to frontend directory**:

   ```bash
   cd frontend
   ```

3. **Install dependencies**:

   ```bash
   npm install
   ```

4. **Start development server**:

   ```bash
   npm run dev
   ```

5. **Build for production**:
   ```bash
   npm run build
   ```

## Usage

- **Development**: Run `npm run dev` to start the development server
- **Production Build**: Run `npm run build` to create optimized production files
- **Preview**: Run `npm run preview` to preview the production build
- **Linting**: Run `npm run lint` to check code quality
